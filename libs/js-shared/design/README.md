## Introduction

`@processpuzzle/design` provides the shared design-system primitives for ProcessPuzzle front-end libraries and applications: tokens, themes, and base styling utilities used across the Low-Code platform.

## Status

This library is currently a scaffold. The public API will grow as the design system takes shape.
