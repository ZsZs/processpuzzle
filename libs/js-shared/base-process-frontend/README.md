## Introduction

`@processpuzzle/base-process-frontend` is the front-end building block of the ProcessPuzzle Process Engine. It provides the Angular constructs to author, monitor, and drive business processes in a Low-Code application. The library complements the [`base-process-backend`](../../java-shared/base-process-backend/README.md) Spring Boot module that executes long-running processes server-side.

## Status

This library is currently a scaffold. The public API will grow as the process engine takes shape.
