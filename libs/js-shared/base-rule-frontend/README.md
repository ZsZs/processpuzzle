## Introduction

`@processpuzzle/base-rule-frontend` is the front-end building block of the ProcessPuzzle Rule Engine. It provides the Angular constructs (services, stores, components) needed to author, evaluate, and visualize business rules in a Low-Code application. The library complements the [`base-rule-backend`](../../java-shared/base-rule-backend/README.md) Spring Boot module that hosts the rule definitions and runs the evaluation server-side.

## Status

This library is currently a scaffold. The public API will grow as the rule engine takes shape.
