## Introduction

`@processpuzzle/base-workflow-frontend` is the front-end building block of the ProcessPuzzle Workflow Engine. It provides the Angular constructs to author, monitor, and drive business workflows in a Low-Code application. The library complements the [`base-workflow-backend`](../../java-shared/base-workflow-backend/README.md) Spring Boot module that executes long-running workflows server-side.

## Status

This library is currently a scaffold. The public API will grow as the workflow engine takes shape.
