# ProcessPuzzle Framework
The capabilities of the ProcessPuzzle Plattform are based on a couple of stand-alone libraries. You can use there libraries independent from the whole platform.
It can help to compile CRUD applications much faster than building it from scratch.
- [Util](/libs/js-shared/util/README.md) - generic purpose functions, useful at building Angular applications
- [Base Entity](/libs/js-shared/base-entity/README.md) - Angular Material Table and Reactive Form generator based on Entity Definition.
- [Widgets](/libs/js-shared/widgets/README.md) - General purpose UI widgets, useful for application development.
- [Auth](/libs/js-shared/auth/README.md) - Authentication and Authorization.
- [Test-Util](/libs/js-shared/test-util/README.md) - Helper functions for unit testing.
- [CI/CD](/.github/README.md) pipeline for all containing building blocks.
